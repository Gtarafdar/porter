# Contributing / iterate elsewhere

1. Clone the repo on the other Mac or pull latest.
2. `npm install && npm run build && npm start`
3. Update [PLAN.md](PLAN.md) checkboxes when you finish a slice.
4. Prefer small commits: one feature or fix per commit.
5. Never commit `~/.porter/` secrets or real pair tokens.
6. Edit `mcp.cursor.json` per machine (absolute path to `packages/core/dist/mcp.js`).
